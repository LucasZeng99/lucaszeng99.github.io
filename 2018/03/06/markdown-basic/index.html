Markdown basic usages

Headings

To write a heading or title in markdown, simply add # in the front.
# your title

## your subtitle

Text body

To type in text, you don’t have to do anything extra.

Bold

To bold a word, add ** to both sides.

**one or more words**

Code block

Quote your codes in two ` sign. (on the top left of your keyboard.)

this is code block

Bash command

You can use ` to wrap up a bash command block.

a ` block can contain multiple lines of codes.
